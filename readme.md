# Project setup

## Backend

```bash
cd backend
npm install
docker compose up -d
npm run start:dev
```

### Run backend project with assets folder

```bash
npx serve -l tcp://127.0.0.1:9000 ./assets
```

## Frontend

```bash
cd backend
npm install
npm run dev
```
